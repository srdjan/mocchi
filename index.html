Semantic HTML, Minimal Classes

sem-mochi leverages modern CSS capabilities and semantic HTML to minimize class usage while maintaining sophisticated styling capabilities.

Semantic Benefit: Cleaner markup, better accessibility, improved maintainability, and enhanced SEO.

The fluid typography system uses clamp() to set minimum and maximum sizes with smooth scaling between them.

Mathematical Fluid Scaling

Mathematical fluid scaling uses a formula to calculate the perfect size at any viewport width:

This text scales from Small to Medium ([fluid-calc="sm-md"])

This text scales from Medium to Large ([fluid-calc="md-lg"])

This text scales from Large to Extra Large ([fluid-calc="lg-xl"])

This text scales from XL to 2XL ([fluid-calc="xl-2xl"])

This text scales from 2XL to 3XL ([fluid-calc="2xl-3xl"])

The formula used is: min-size + (max-size - min-size) * ((viewport-width - min-viewport) / (max-viewport - min-viewport))
